CSS3 Animations

Jitter Animation

@keyframes jitter {
	0% {
		top: -2px;
		left: -2px;
	}

	50% {
		top: 2px;
		left: 2px;
	}

	100% {
		top: 0px;
		left: 0px;
	}
}

Flashing Animation

@keyframes flashing {
	from {
		background: #f1c40f;
	}

	to {
		background: none;
	}
}

Applying the Animation

nav ul li:hover {
	-webkit-animation: jitter 150ms infinite;
}

Don't Forget the Browser Specific Prefixes!

@-webkit-keyframes flashing {
	...
}

@-moz-keyframes flashing {
	...
}

@keyframes flashing {
	...
}

nav ul li:hover {
	-webkit-animation: ...;
	-moz-animation: ...;
	animation: ...;
}
